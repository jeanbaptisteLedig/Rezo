# Rezo
Social Network React

## Requirement
- node v11.2.0
- npm v6.4.1
- yarn v1.15.2
- mongo v3.4.3

```bash
git@github.com:jeanbaptisteLedig/Rezo.git
cd Rezo
```


## Run frontend
```bash
cd frontend
npm install
npm start
```

## Run backend API (mongodb should be started)
```bash
cd backend
npm install
nodemon app
```
```bash
mongod //to start mongodb
```
